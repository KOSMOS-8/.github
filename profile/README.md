## 🎯 The Problem · 我们要解决的问题

Both sides of a venture conversation waste enormous time before they can tell whether the meeting is worth having.

创投双方在见面前，都无法判断"这次会面值不值得发生"——而判断之前，大量时间已经浪费。

| For Founders · 对创业者 | For Investors · 对投资人 |
| --- | --- |
| Met many investors, few led anywhere. | ~95% of inbound BPs are dropped after the first call. |
| Re-pitching the same progress to everyone, over and over. | No way to know if a founder is worth meeting — but skipping means flying blind. |
| No idea what a given VC actually cares about before the meeting. | Too many hours sunk into low-quality first conversations. |

**The root issue is one thing seen from two sides: neither party can judge, *before* meeting, whether the meeting is worth it.**

**本质是同一件事的两面：双方在见面前都无法判断这段关系是否值得推进。**

## 🧭 The North Star · 我们的唯一终点

> **Founder initiates an NDA → Investor signs → the original BP is unlocked.**
>
> **Founder 发起 NDA → Investor 签署 → BP 原文解锁。**

When this happens, the platform has built enough trust between the two sides: the founder is willing to protect their core materials, the investor is willing to take on confidentiality, and both believe the relationship is worth advancing. Whether funding ultimately closes is *beyond* this milestone — trust is the product.

走到这一步，代表平台已在双方之间建立了足够的信任。融资是否成功，不在这一终点的范围内——**我们交付的是信任本身。**

## ⚙️ How It Works · 核心能力

1. **Material Upload · 资料上传** — Founders upload BP, résumé and supporting docs; investors upload investment thesis and portfolio. The data foundation everything else runs on.
2. **Agent‑Built Profiles · 智能体建档案** — An agent reads the uploads and asks two layers of follow-ups: generic gaps first, then sector-specific deep dives. *Profile quality drives match quality — the question library is the moat.*
3. **BP Access Control · BP 权限控制** — The agent publishes a redacted summary while the original stays locked. Founders set the information boundary: what the agent may answer, and what must wait for a real conversation.
4. **Two‑Layer Matching Engine · 双层匹配引擎** — Layer 1 rule-based filtering (sector, stage, geography, check size); Layer 2 semantic matching (BP × thesis embeddings) → Top‑N candidate pairs.
5. **Auto‑Built VC Agents · VC Agent 自动构建** — The system assembles an initial VC persona from public sources (website thesis, portfolio, interviews) — *no active effort required from scarce investor time.*
6. **Pre‑Conversation · 预对话机制** — Agents stress-test the business logic and surface what each side truly cares about. The system then generates summaries: **Red Flags + a meeting-prep checklist** for the founder, **follow-up hotspots** for the investor.
7. **Mutual Like · 双向确认** — Only when *both* sides accept does the relationship advance.
8. **Shared Workspace · 共享工作区** — A 1:1 room where the NDA is signed, the BP is unlocked, and meetings are booked via Google Calendar.
9. **Permit Authorization · Permit 授权** — After signing, an investor can request a Permit; once granted, their agent auto-tracks the founder's progress summaries — so founders never re-pitch the same update. Revocable anytime.
