# VC‑Match

### KOSMOS8 — 创投圈的 AI 预匹配平台

*在真人见面之前，让 Founder 与 Investor 通过 AI 完成筛选与准备，只在真正值得的时候才见面。*

</div>

---

## 🎯 我们要解决的问题

创投双方在见面前，都无法判断"这次会面值不值得发生"——而判断之前，大量时间已经浪费。

| 对创业者（Founder） | 对投资人（Investor） |
| --- | --- |
| 见了很多投资人，大多数没有实质推进。 | 每周收到大量 BP，约 95% 首轮沟通后即被淘汰。 |
| 要反复向不同投资人重复汇报公司进展。 | 见面前不知道值不值得见，但不见又不知道。 |
| 不知道某个 VC 真正在意什么，见面前无法准备。 | 大量时间花在低质量的初步沟通上。 |

**本质是同一件事的两面：双方在见面前都无法判断这段关系是否值得推进。**

---

## 🧭 我们的唯一终点

> **Founder 发起 NDA → Investor 签署 → BP 原文解锁。**

走到这一步，代表平台已在双方之间建立了足够的信任：Founder 愿意保护自己的核心资料，Investor 愿意承担保密义务，双方都认为这段关系值得推进。融资是否成功，*不在*这一终点的范围内——**我们交付的是信任本身。**

---

## ⚙️ 核心能力

1. **资料上传** — Founder 上传 BP、简历等材料；Investor 上传投资 Thesis、Portfolio。这是后续所有机制的数据基础。
2. **智能体建档案** — Agent 读取上传资料后进行两层追问：先补通用缺失项，再做赛道专属深度追问。*档案质量决定匹配质量，问题库就是护城河。*
3. **BP 权限控制** — Agent 生成脱敏摘要对外展示，原文锁住。Founder 设定信息边界：哪些问题 Agent 可以回答，哪些引导至真人见面。
4. **双层匹配引擎** — 第一层规则过滤（赛道、阶段、地域、Check Size）；第二层语义匹配（BP × Thesis 的 Embedding 相似度）→ 输出 Top‑N 候选匹配对。
5. **VC Agent 自动构建** — 系统自动抓取 VC 公开信息（官网 Thesis、Portfolio、访谈）生成初版画像——*无需占用稀缺的投资人时间主动配合。*
6. **预对话机制** — 双方 Agent 互相压力测试商业逻辑，挖出彼此真实关注点。结束后系统生成摘要：Founder 侧得到 **Red Flags + 见面准备清单**，Investor 侧得到**追问热点**。
7. **双向 Mutual Like** — 只有双方都 Accept，关系才进入下一步。
8. **共享工作区（Shared Workspace）** — 1 对 1 聊天室，在这里签 NDA、解锁 BP、并通过 Google Calendar 约会议。
9. **Permit 授权** — Investor 签完 NDA 后可申请 Permit，授权后其 Agent 自动追踪 Founder 的动态摘要——Founder 不必再重复汇报进展，且可随时撤回。

## 🚶 用户路径

```
Founder                                   Investor
───────                                   ────────
登录选身份                                  登录（VC Agent 自动构建）
上传资料                                    确认 / 修正画像与偏好
Agent 追问 → 建立档案                        收到项目推荐
设定 BP 信息边界                            与 Project Agent 预对话
收到投资人推荐                               ↓
与 VC Agent 预对话                          ↓
        └──────────►  双向 Mutual Like  ◄──────────┘
                          │
                   进入 Shared Workspace
              签 NDA → BP 解锁 → 约会议
                          │
                   Permit 授权与动态追踪
```

## 👥 核心用户

- **Founder** — 正在融资或希望拓展创投圈人脉的早期创业者。
- **Investor** — 寻找优质项目或希望认识创业者的 VC、Angel Investor。
- **Founder ↔ Founder** — 按赛道相似度与人格特质匹配的同行社交（无 NDA、无 BP，纯社交）。
